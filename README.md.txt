# Truck Check-In System

## Overview

A warehouse driver check-in application that allows:

- Driver self check-in
- License scanning
- Daily load schedule imports
- Shipping office dashboard
- Printable load checklists

## Technology Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Java
- Spring Boot

### Database
- PostgreSQL

## Planned Features

- CSV schedule uploads
- Driver license OCR
- PDF417 barcode scanning
- Driver check-in portal
- Shipping office dashboard
- PDF checklist generation
